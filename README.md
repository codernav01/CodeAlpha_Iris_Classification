# Iris Flower Classification 🌸

This project is part of the **CodeAlpha Data Science Internship**.

## Project Overview
The goal of this project is to classify Iris flower species using machine learning.  
The model predicts whether the flower belongs to **Setosa, Versicolor, or Virginica** based on flower measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model
Logistic Regression

## Dataset
The dataset is provided by **Scikit-learn's built-in Iris dataset**.

## Project Workflow
1. Import required libraries
2. Load Iris dataset
3. Convert dataset into DataFrame
4. Data visualization
5. Train-test split
6. Train Logistic Regression model
7. Evaluate model accuracy

## Result
The model successfully classifies Iris flower species with high accuracy.
